The Honeynet Project

GSoC 2018 Work Product

DroidBotRecorder is an Android App, built for automating the input to DroidBot an automated testing tool for Android.

Features Implemented

  • Can record touch interaction with an android emulator
  • Can serialize the interaction in the form of a UI Transition Graph (in json)

Work in progress

  • Docker support is an important feature, and is almost complete, but does not work at the moment.

    • Root access is forbidden on butomo1989/docker-android which is an important requirement for DroidBotRecorder

    • I'm working on creating a docker image myself, but it currently does not work.
